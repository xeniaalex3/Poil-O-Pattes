## Install the project 

```
# Clone the repository
$ git clone git@github.com:O-clock-De-Vinci/projet-poil-o-pattes-front.git

# Access the project folder at the command prompt
$ cd projet-poil-o-pattes-front

# Install the dependencies
$ yarn

# Run the script 
$ yarn start

```
